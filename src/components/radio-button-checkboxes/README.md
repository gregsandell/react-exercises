# radio-button-checkboxes
Reqct component + CSS to create radio button style checkboxes.  The component is in `src/RadioButtonCheckboxes.jsx` and the CSS is in `src/RadioButtonCheckboxes.css`.  The component takes two props: `options` (an array of strings) and `selectedOption` (a string).  It renders a checkbox for each option, with the one matching `selectedOption` preselected.  When a checkbox is clicked, it calls the `onChange` prop with the new selected option.
