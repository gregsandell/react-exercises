# Fruits

This was a challenge presented by Ace Fortuna for a role with
[https://www.rollcredits.io/](https://www.rollcredits.io/) in January 2026.

Currently (Jan 2026) the data api hosted by rollcredits.io is still
available.  If they shut it down or the API key ceases to work, the
api can be faked using the data in `fruits/respose.ts`.

## Original Instructions
Build a simple React + Typescript application that displays a
table of fruits with different attributes. The user should be able
to type into various inputs to filter the table by fruit name,
primary color & weight in grams. They should be able to search
by multiple attributes at once.

Notes:
* Create a fruit interface to model the data from the api route
* Use raw jsx to render the table
* Here is the spec for the API: [Fruit Market API](https://frontend-interview-api.rollcredits.io/) 
* APl key: sk-fruit-api-123456789@abcdef
